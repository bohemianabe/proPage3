About Me

Welcome to my page, my name is Abel - a web developer in the DC area. I'm passionate about design and good reusable code, as well as understanding data. I often blog on understanding and manipulating data using python. It's an incredible tool to use, and gain insight on any project you're working on. For a few years now I've dabbled with font end code such as html & CSS, but never found the right time to dedicate myself completely to all aspects of web development. That all changed due to the pandemic (still a little weird writing that). I've since completed a bootcamp as a full stack developer, and data science.

I see coding as a great adventure, and finding this professional field is like finding home. I appreciate you visiting my site, and I hope we can work together soon.

Ignite -Game Search Engine

I hope you like gaming, and I hope you are in need of a search engine to look up your next game. Ignite is a React App that does just that built with axios for an API call, and redux to manage the states.
